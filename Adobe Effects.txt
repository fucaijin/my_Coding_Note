选中现有图层，然后画图，效果是给那个图层画蒙版，画的图形蒙版内的内容才能看见，否则当前图层的其他区域都看不见

空格 + 鼠标左键  → 移动图形
shift + 鼠标左键/滚轮？ → 垂直绘画(应该是垂直方向移动)
shift + alt + 鼠标左键/滚轮？ → 在当前图层缩放图形

在一个图层中画多个图形 → 选中内容，然后画图即可，然后每个图形都可以设定自己的旋转缩放等属性，只控制那个图形自己，如果要改变图层的属性，那么图层中所有图形效果都会改变
Ctrl + T → 选中图层中的图形，按下Ctrl + T就可编组

倒角 ： 图层 → 图形 → 路径 → 内/外圆度 (应用范例，例如把利用多边形五角星变成花朵，只要改变内外圆度即可)

画圆 → 画椭圆/或者双击椭圆工具 → 选择填充颜色 → 修改图形参数 → 内容 → 椭圆 → 椭圆路径 → 解锁路径绑定 → 修改x和y的值为一样，然后重新绑定即可

阵列工具 → 选中内容右侧的添加 → 中继器 → 修改中继器的参数/副本个数就是要阵列的个数

控制路径/线条长度 → 选择内容 → 添加 → 修建路径 → 修改修建路径参数，改变修建路径的开始和结束即可改变路径的长度（此处的偏移工具，是在路径上偏移开始和结束点，会产生线条中断/水管流水的效果）

心电图效果/波浪效果 → 画直线 → 选中内容 → 添加 → 摆动路径 → 调整参数 → 大小调的是高度 → 详细信息调的是波浪之间的间隔/密度

Z字形 和摆动路径很相似， 也可以做波浪效果，但是波浪是均匀的 

内容中的效果排列顺序很重要，后面的效果都是在前面的效果上完成的，例如需要字体按某个路径排列，就得先画路径，把字的层放在路径下面

布尔/相加相减 → 选择内容 → 添加 → 合并路径 → 修改内容下的合并路径的参数(相加相减)

定格关键帧：就是图形运行到那个关键帧就停止，直到运行到下一个关键帧再直接跳到那个关键帧 → 选取关键帧，然后右键 → 切换定格关键帧即可

将直线运动路径变为曲线路径 → 选取开始和结束中间的一个时间点拖动图形到合适的位置，然后选中要更改的关键帧/也可以选择全部 → 右键 → 关键帧插值 → 空间插值选择贝塞尔 → 拖动路径的手柄使直线变为曲线

一下选中所有关键帧，并按比例缩短帧的时间 → 展开图形 → 选择位置(此时选中了位置的所有关键帧) → 按住Alt键 → 拖动最后一个关键帧即可实现整体所有关键帧的时间的缩放

动画慢入慢出 → 选中关键帧 → 右键 → 辅助 → 关键帧辅助 → 缓入/缓出

右下侧的眼睛旁边那个就是速度等曲线的调整

如果所有图形都预合成在了一个图层中，那么可以点击图层栏的顶栏“图层名称”右键 → 列数 → 伸缩 → 更改拉伸因素百分比来整体缩放动画时间，例如50%就是缩短时间为原来1半

左侧点击图层 → 右键 → 时间 → 启用时间重映射 → 右下角的两个关键帧代表动画的开始和结束，如果缩短这两个关键帧 ，动画速度就会变快

截断动画/结束动画剪刀 → 选中图层 → 拖动时间竖轴到想要结束的地方 → Alt + ]  结束动画

截取动画 → 如果只想要动画的一半，就更改启用时间重映射时候的那个100值为50，动画就会执行到一半就停止。如果要正放动画，然后倒放动画，就设置开始关键帧为0，中间关键帧为100，最后关键帧为0

螺旋运动动画 → 更改锚点 + 旋转即可实现

物体波浪形运动 → 更改锚点 + 位置即可实现(位置在匀速直线运动，锚点一直在上下运动)

按住Shift 然后再绑定父对象，就会把当前的对象的中心点直接移动到父对象的中心点重合

空对象 → 在图层面板右键 → 新建 → 空对象 → 把所有图层都将空对象作为（同时选中多个图层，然后指定其中一个过去即可）父对象 → 更改父对象的设置 ，就会整体的对其他的子对象产生影响（注意删除空对象会导致动画还是空对象设置的动画，所以删除前要恢复回空对象前的动画状态）

小苍蝇飞的步骤 → 在图层面板右键 → 新建 → 空对象(视频中是纯色也一样) → 顶部标题栏的窗口 → 动态草图 → 右侧动态草图的设置 → 线框(降低CPU的使用率) → 开始捕捉 → 左键鼠标移动刚刚新建的空对象 → 松开右键就是结束捕捉 → 选中空对象的位置(即选中所有位置的关键帧) → 窗口 → 平滑器 → 调整平滑器的容差值 → 应用即可平滑路径 → 拖动虫子图层进来，然后设定空对象为虫子的父关系(拖动的时候按住shift，虫子的中心就会直接以空对象为中心重合过去) 
→ 然后修改 小苍蝇的旋转角度，在不同的位置创建不同的关键帧，旋转小苍蝇，使之变得和飞行反向一致 → 隐藏白色空对象即可
增强 (虚线路径)： 新建一个形状 → 在内容添加路径 → 把上面的空对象的位置关键帧都复制(编辑 → 复制)到新建形状的路径下，如果有偏差就修改一下该形状的位置（变换 → 位置）改为0即可 → 对路径描边 → 添加虚线 → 修剪路径 → 根据苍蝇飞到的地方添加不同的修建关键帧即可实现苍蝇飞到哪，路径就跟到哪

表达式：
按住Alt键 点击关键帧(例如位置)旁边的小闹钟 → transform:position修改为：loopOut("Cycle") 即可让动画无线循环(loopOut("PingPong")也是一样的效果)
loopOut("Continue") 例如让图形继续按原来的地方飞行

小正方形向前跳动 → 图形本身有上下跳动的关键帧 → 创建空对象，空对象的是向前移动的关键帧，然后空对象作为父对象绑定图形即可

loopOut("Cycle")和loopOut("offset") offset针对的是多个帧进行循环 Cycle是指针对前后两个帧进行循环

遮罩 → 在图层区域 右键 → 纯色 → 调整纯色图层大小 → 设置另一个图层为Alpha（就是模式旁边那个） → 即可实现只有另一图层出现在纯色区域的时候才显示
